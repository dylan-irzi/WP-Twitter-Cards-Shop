== Description ==

WP Twitter Cards Shop is a WordPress plugin allowing automatic way to add twitter cards, fully compatible with WooCommerce.

Features include:

*   Simple, Easy to install, you can get "Twitter Cards" WooCommerce your store immediately.
*   Social, Your Twitter followers will see the information of your most detailed and easy to read product.
*   Control, you can adjust the plugin to your needs.

== Installation ==

1. Install WP Twitter Cards Shop either via the WordPress.org plugin directory, or by uploading the files to your server.
2. After activating WP Twitter Cards Shop.
3. Go to settings and place your twitter username.
4. That's it.

== Frequently Asked Questions ==

= I have a WooCommerce Store is compatible? =

Yes , WP Twitter Cards Shop, is fully compatible with WooCommerce.
